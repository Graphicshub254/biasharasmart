# BiasharaSmart

KRA tax compliance and M-Pesa reconciliation platform for Kenyan SMEs.

## Setup

1. Install dependencies:
   ```bash
   yarn install
   ```

2. Configure environment:
   ```bash
   cp .env.example .env
   ```

3. Build the project:
   ```bash
   yarn build
   ```

## Structure

- `apps/api`: NestJS backend
- `apps/mobile`: React Native placeholder
- `packages/shared-types`: Shared TypeScript interfaces
